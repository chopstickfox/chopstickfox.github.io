    Summary

    Aleksandr Dozorov and I created a Factory that produces Aircraft in Java. Classes Helicopter and Airplane implement Supplier’s method material(), which return material the aircraft made out of, as well as a method garantyYears(), which returns guarantee years on the aircraft. The class Aircraft has sets names, quantity and returns if an Aircraft was tested. The class Airplane extends Aircraft and sets type. Helicopter extends Aircraft and sets maximum carrying capacity. The class Factory contains all the aiplanes produced. The class Factory also has a method setTechnicalName() to remove the nickname, and also has a method removeOldAndProt() which removes prototypes and untested aircraft.
